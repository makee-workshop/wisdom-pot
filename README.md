# Makee.io wisdom-pot workshop

## Philosophy

在兩天的工作坊中，你將學得：
* 認識Arudino Yún平台及IOT開發架構；
* 軟體：Arduino語法、OpenWrt、Bridge、Python等；
* 雲服務：Restful、Websocket、WoT等；
* 硬體：電子電路基礎、感測器及馬達操作等；
* 實作：聯網智慧盆栽
